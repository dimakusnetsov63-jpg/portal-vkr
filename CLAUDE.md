# CLAUDE.md

Этот файл содержит контекст для Claude Code при работе с этим репозиторием.

## О проекте

Next.js приложение (App Router) с TypeScript и Tailwind CSS.

## Стек

- Next.js 16 (App Router, Turbopack)
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint

## Структура

- `src/app/` — страницы, layout и роутинг (App Router)
- `public/` — статические файлы

## Команды

- `npm run dev` — dev-сервер
- `npm run build` — production-сборка
- `npm run start` — запуск собранного проекта
- `npm run lint` — ESLint
