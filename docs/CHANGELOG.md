## 2026-07

### Added

- **Портал на модульной структуре** — разделы (Обзор, Потребность, Кандидаты,
  Описание вакансий, Маркетинг, Аналитика, Уведомления, Настройки) с общими
  UI-примитивами и единым `PortalContext`.
- **Подключение Supabase** — клиент/сервер/middleware, env через
  `NEXT_PUBLIC_SUPABASE_*`, генерируемые типы.
- **Раздел «Кандидаты» на реальных данных** — таблица `public.candidates`
  (миграция `20260721133910`), RLS-политики для `authenticated`
  (`20260721202609`), CRUD и архивирование через `archived_at`.
- **Аутентификация сотрудников** — вход по email+паролю, восстановление пароля
  (`/forgot-password`, `/update-password`), защита роутов через `proxy.ts`
  (коммит `41e4ade`).
- **Управляемые справочники кандидатов** — таблица `candidate_list_options`
  (recruiter/manager/coordinator/city) с порядком и деактивацией
  (миграции `20260722213735`, `20260722214949`; коммиты `f78795d`, `f553e22`).
- **Аналитические карточки кандидатов** — «Ожидают выхода» / «Успешно вышли» /
  «Есть медкнижка» по текущей отфильтрованной выборке (коммит `a23e7bc`).
- **Проектная документация** — `CLAUDE.md`, `docs/`, README раздела кандидатов.

### Changed

- **Рефакторинг раздела «Кандидаты»** — вынос в папку `sections/candidates/`,
  отделение чистой логики (`candidateFilters.ts`, `candidateMetrics.ts`),
  разбиение компонента (`CandidatesTable`, `AddCandidateModal`); поведение и
  дизайн без изменений (коммит `76616ed`).
- **Ребрендинг в «ВКР — Ваш кадровый ресурс»** — название, логотип и favicon;
  замена прежнего брендинга (коммит `73fda60`).

### Fixed

- **Продакшн-конфигурация auth** — финализация настройки и удаление
  диагностического кода (коммит `7abd457`).
- **Чтение `NEXT_PUBLIC_SUPABASE_*`** — переход на статический `process.env.X`,
  чтобы переменные инлайнились в клиентский бандл (коммит `e509b98`).
